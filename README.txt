#  Overview
This is intended to provide functions to allow

* an admin to create a sign up form with optional fields and to display all those signed up and what a logged in user is signed up to.
* for a user to subscribe or unsubscribe to something
* to list all those signed up for a specific event
* to list everthing to which a user is signed up

# Installation
1. Upload `members-signup` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
